## Git Facts

by ____Sora Ito_____

git is a kind of _____version____ control software, similar to the "track changes" feature of Microsoft Word or Google Documents.

git tracks changes to files in a special folder called a __repo__, which is short for ____repository____.

git keeps a log of all the changes you have made to your files. Each change is called a ____commit___. You always provide a short comment describing what happened in the change.
